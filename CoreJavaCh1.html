Java设计目标

  1. 简单性：C++ --
  2. 面向对象：和C++主要的不同在于用接口代替多重继承、运行时自省
  3. 分布式：丰富的例程库
  4. 健壮性：编译时可以检测出一些错误
  5. 安全性：设计时就能防范错误——运行时堆栈溢出、破坏自己进程以外的内存、未经授权读写文件
  6. 体系结构中立：利用虚拟机实现指令集无关性
  7. 可移植性：数据类型具有固定大小、标准unicode存储等
  8. 解释性：解释器的实现
  9. 高性能：即时编译带来的即时优化
  10. 多线程：第一个支持并发设计的主流语言
  11. 动态性：库中可以自由条件新方法和实例变量
  12. applet

一次编写，随处运行

Java8特性：lambda表达式，包含默认方法的接口，流和日期/时间库

Java基本程序设计

最基本

public访问修饰符

类名：必须以字母开头，后面接数字或者字母，不能使用保留字

推荐类名：大写字母开头，每一个单词首字母大写

必须包含main方法才能执行

注释

xxxxxxxxxx

数据类型

整型：int、short、long、byte

浮点：float、double（特殊值：Double.POSITIVE_INFINITY、Double.NEGATIVE_INFINITY、`Double.NaN）

char和Unicode：最好使用String

boolean：不能和整型相互转换

变量

每个变量都有类型，由有意义的unicode字符构成，逐一声明变量可以提高程序的可读性。

变量必须初始化之后才能使用，且声明尽可能的靠近第一次使用的地方。

常量

final指示常量，只能被赋值一次，习惯上常量名全大写。

当希望某个常量可以在一个类的多个方法中使用时，用static final设置类常量，其定义位于main的外部。

！虽然const为保留字，但并没有使用，Java使用final。

运算符

！整数被0除会产生异常，浮点数会得到无穷大或者NaN。

如果需要严格的浮点计算来避免截断影响精度，在main函数之前使用strictfp。

数学函数

最常用：Math.sqrt();Math.pow();

余数为正数：floorMod;

常用三角函数：Math.sin;Math.cos;Math.tan;Math.atan;Math.atan2;

常用指数相关函数：Math.exp;Math.log;Math.log10;

常用近似值：Math.PI;Math.E;

强制类型转换：类型转换的顺序（略）；

结合赋值；自增自减；关系和boolean；三元运算；

位运算符：&and,|or,^xor,~not;（布尔值也可以计算）左移右移；

枚举类型：只能是声明中给定的类型或者null值。

字符串

就是Unicode字符序列。

substring方法：从一个较大的字符串中提取出一个子串。

拼接：加号拼接；或者静态方法String.join。

不可变字符串：String类没有提供修改方法，若要修改，提取需要的字符，再拼接上替换的字符串。

虽然效率不高，但是编译器可以让字符串共享。

！和C++的char数组有本质的不同。

比较字符串：

  1. 使用equals
  2. 不区分大小写：使用equalsIgnoreCase
  3. 不能使用==，因为==只能比较两个字符串是否在同一位置上。

空串的检查：if(str.length()==0)或if(str.equals(""))

null串的检查：if(str==null)

既不是null也不为空串：if(str!=null&&str.length()!=0)

码点：实际的char

代码单元：utf-16中的单元

length返回代码单元数。

codePointCount返回码点数。

charAt返回指定代码单元。

流程控制

块（block）：由大括号括起来的Java语句。

Java不能在嵌套的两个块中声明同名变量，而C++可以。

if、if-else、for（略）。

for循环的一条不成文的规定：3个部分应当对同一个计数器变量进行初始化、检测、更新。

！以浮点数作为检测条件需要考虑截断影响了精度进而影响判断。

switch语句的case标签可以是：char、byte、short、int的常量，枚举常量，从JavaSE7开始还可以是字符串常量；使用枚举值时可以不必指明枚举名。

带标签的break：标签必须放在希望跳出的最外层循环之前，并且必须紧跟一个冒号；只能跳出语句块，不能跳入语句块。

continue：将控制转移到最内层循环的首部；while中跳到检测，for中跳到更新部分。

带标签的continue：跳到与标签匹配的循环首部。

数组

声明和初始化是分开的：

xxxxxxxxxx
int[] a;
a = new int[100];

创建数字数组时初始化为0，boolean数组初始化为false，对象初始化为null；

获得元素个数：length。

一旦创建，就不能改变大小。

for-each循环

xxxxxxxxxx

  System.out.print(e);

另外一个打印所有值的方法：

xxxxxxxxxx
System.out.print(Arrays.toString(a));

数组的初始化和匿名数组

一般的初始化：

xxxxxxxxxx
int[] a = {0,1,2};

初始化一个匿名的数组，可以在不创建新变量的情况下重新初始化一个数组：

xxxxxxxxxx
a = new int[]{0,1,2,3};

允许长度为0的数组，可以作为返回值，不同于null

xxxxxxxxxx
return new int[0];

数组拷贝

拷贝的同时也可以增加数组的大小：

xxxxxxxxxx
int[] b = Arrays.copyOf(a,a.length);
b = Arrays.copyOf(a,a.length*2);

如果是数值型，多余的被赋值为0，如果是布尔型，多余的为false。

Java数组相当于C++分配在堆上的数组指针。
